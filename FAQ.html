Got questions?

I'm I able to use your mod in my modpack?

Yes you can, but you'll need to give credits to the CurseForge download or to this webpage.

Why should I donate you?

Well, you are not obligated to donate me but if you want to theres a link to my PayPal, the donations will be used mainly to pay the domain for this WebPage and the time spent modding.

How many developers are working on these mods?

Theres just 1 developer (me, Twelve a.k.a Hispano / zk12) I'm doing all the coding, model making, animations, textures, etc. I talk as "us" in all this WebPage because I take in consideration my moderators of the discord server as they're helping me a lot :-)

Are you gonna export the mods for more MC versions?

For now is impossible for me since is takes a lot of time to develop these mods for a single MC version, i couldn't imagine doing it for all of them.

Why MC 1.19.2?

Mostly because MCreator (the software i use to develop these mods) works better with this game version.

Why using are MCreator?

Well, I'm not that good at coding so MCreator helps me a lot learning coding in java, basically these mods are made mainly to help me learn java :-)
