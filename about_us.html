Welcome to Twelve-k-Productions

Let's talk a bit about our project and what we want to do with it.. First, our project is meant to delivery High-Quality mods for minecraft users, we will only deliver these mods in a single version of minecraft as if we deliver the mods for all the versions we wont finish (version: 1.19.2)

Second, the mods will be patched along the time but don't except from us to solve all the problems since we are not experts but we will always try to patch them..

Third-ish thing, regards the First thing mentioned here, when all our mods are done and patched in the best way for the version 1.19.2 we will try to gather our mods for more versions but this wont be 100% guaranteed.
